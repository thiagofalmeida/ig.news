## Building

You'll need [Node.js](https://nodejs.org) and [Yarn](https://yarnpkg.com/) installed on your computer in order to build this app.

```bash
git clone https://github.com/thiagofalmeida/ig.news.git
$ cd ig.news
$ yarn install
$ yarn dev
```
